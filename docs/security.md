# Segurança

Plataforma foi projetada para uso interno controlado.

## Controles ativos

- Scripts desligados por padrão.
- Preview separado.
- Caminhos ZIP validados.
- Sem autenticação externa.
- Sem envio automático de arquivos.
- Sem banco remoto.

## Confiança necessária

Canvas permite mesma origem para edição visual completa. Código importado deve vir de fonte confiável. Ativar JavaScript concede execução dentro do navegador.

Não importe projetos desconhecidos.
